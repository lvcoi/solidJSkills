# StartClient

[Edit this page](https://github.com/solidjs/solid-docs/edit/main/src/routes/solid-start/reference/client/start-client.mdx)

`StartClient` is a component that wraps the application root. It can be passed to [`mount`](start-client.md) in the [`entry-client.tsx`](../entrypoints/entry-client.md).

```


import { StartClient } from "@solidjs/start/client";
```
